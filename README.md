# AngularJS MVC Tabbed Demo

This repository now contains a simple AngularJS MVC app with:

- Horizontal tabbed navigation
- Six linked example pages
- Persistent footer
- Fully commented HTML, CSS, and JavaScript source for learning purposes

## Run Locally

Use any static server and open the app in a browser.

### Option 1: Python (usually preinstalled)

```bash
cd /workspaces/angularjs-todomvc
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

### Option 2: VS Code Live Server

Open `index.html` with Live Server.

## App Files

- `index.html` - App shell and AngularJS templates
- `styles.css` - Visual styling for tabs, content, and footer
- `app.js` - AngularJS module, model service, route config, and controllers
